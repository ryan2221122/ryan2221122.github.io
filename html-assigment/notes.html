HTML Tags Description List

<p>
The <p> tag defines a paragraph in HTML.
<a>
The <a> tag defines a hyperlink to another webpage or resource.
<img>
The <img> tag is used to embed images in an HTML document.
<ul>
The <ul> tag defines an unordered list.
<table>
The <table> tag is used to create a table in HTML.
